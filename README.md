# unstructured-excel-to-2d

Reusable GitHub Copilot skill for converting messy, non-2D business Excel workbooks into IT-ready 2D tables with Python.

## Includes

- Workspace skill definition in `.github/skills/unstructured-excel-to-2d/`
- English workflow guidance
- Chinese guide for business and delivery teams
- Before/after transformation demo
- Validation checklist
- Reusable Python transform and validation templates

## Main Use Case

Use this skill when business provides irregular Excel files with multi-row headers, matrix-style metrics, inconsistent naming, derived-field requirements, and a need to prove transformed outputs match the source.
